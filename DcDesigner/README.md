# DcDesigner 关卡制作工具

### 程序入口：  `DcDesigner.py`

## 需要PYQT5依赖 
## ubuntu下播放视频有声音无音频
## window需要安装解码器，例如: `K-Lite`

#### 正在实现姿势检测功能。。。

#### 关卡信息文件格式：
```
{
    "videopath": ""
    "info":
        [
            {
                "timeid": 1232,
                "reactime": "0.5",
                "score":  10,
                "posepoints": []
            },
            {},
        ]
}
```
